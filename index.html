How it works

1

Step 1 — Copy the link

Open any supported platform — YouTube, Twitch, Instagram, TikTok, and more. Copy the URL from your browser.

2

Step 2 — Paste the link

Launch the app and paste the link into the input field. UMD will instantly detect the source.

3

Step 3 — Choose format & quality

Select your desired format (MP4, MP3, etc.) and quality (1080p, 720p, audio bitrate).

4

Step 4 — Download

Click the download button. UMD will fetch the media and save it to your device.

How to install?

Windows

Step 1 — Install Deno

irm https://deno.land/install.ps1 | iex

Step 2 — Install FFmpeg

winget install ffmpeg

Step 3 — Clone & Setup

git clone https://github.com/Magerko/universal-media-downloader.git
cd universal-media-downloader
python -m venv .venv
.venv\Scripts\activate

Step 4 — Install & Run

pip install -r requirements.txt
python main.py

macOS

Step 1 — Install Deno

curl -fsSL https://deno.land/install.sh | sh

Step 2 — Install FFmpeg

brew install ffmpeg

Step 3 — Clone & Setup

git clone https://github.com/Magerko/universal-media-downloader.git
cd universal-media-downloader
python3 -m venv .venv
source .venv/bin/activate

Step 4 — Install & Run

pip install -r requirements.txt
python3 main.py

Linux

Step 1 — Install Deno

curl -fsSL https://deno.land/install.sh | sh

Step 2 — Install FFmpeg

Debian / Ubuntu:

sudo apt install ffmpeg

Fedora:

sudo dnf install ffmpeg

Arch:

sudo pacman -S ffmpeg

Step 3 — Clone & Setup

git clone https://github.com/Magerko/universal-media-downloader.git
cd universal-media-downloader
python3 -m venv .venv
source .venv/bin/activate

Step 4 — Install & Run

FAQ

What platforms are supported?

YouTube, TikTok, Instagram, VK, RuTube, Twitch, Kick, Kinopoisk, and many more.

What video quality and formats are available?

Quality settings are configured individually per platform. Supported formats include MP4, WEBM, and MP3 for audio extraction.

Does the app support subtitles?

Yes. You can download subtitles in English, Russian, and Ukrainian.

What languages does the app support?

The app interface is available in English, Russian, and Ukrainian.

Can I download audio only?

Yes. The app supports audio extraction and converts it to MP3 at 192kbps.
